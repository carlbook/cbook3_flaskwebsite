This website is my introduction to HTML, CSS, and Javascript. It is hosted on pythonanywhere and uses Flask and MySQL.

The Digits Recognition page steps through a few ideas in machine learning using the scikit-learn digits dataset, including k-Nearest Neighbors, clustering, and a convolutional network using Keras and Tensorflow. The app on the Stock Chart Practice page is mostly Javascript with AJAX to dynamically generate and modify visual elements.
